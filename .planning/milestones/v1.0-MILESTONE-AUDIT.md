## Critical Breaks (Must Fix)

### 1. Quote ID Field Mismatch (BOOK-01, BOOK-02, BOOK-03, UI-09, UI-10)
**File:** `frontend/js/checkout.js` line 471
**Issue:** `state.quote._id` should be `state.quote.quoteId`
**Impact:** Every booking attempt fails with 400 — no bookings can complete

### 2. vercel.json CORS Misconfiguration (INFRA-01)
**File:** `vercel.json`
**Issue:** Global `Access-Control-Allow-Methods: GET, OPTIONS` blocks POST routes; global `Cache-Control: public, s-maxage=3600` caches dynamic responses
**Impact:** Browser preflight blocks POST to /api/quote and /api/book from GitHub Pages

### 3. Upsell Config Wrong Prices (UPSELL-01, UPSELL-02)
**File:** `lib/upsells-config.js`
**Issue:** 4 of 5 prices differ from spec; airport descriptions reference LAX instead of DEN
**Impact:** Guests see wrong prices; Sebastian gets wrong upsell amounts
